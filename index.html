HTML Cheat Sheet

A quick reference for essential HTML elements and best practices.

Headings

Use headings to structure your content from most important (<h1>) to least (<h6>). Only one <h1> per page is recommended.

<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection</h3>

Text & Formatting

Text can be styled using semantic tags for emphasis or structure. Avoid using them only for looks.

<p>This is a paragraph.</p>
<strong>Important text</strong>
<em>Emphasized text</em>
<mark>Highlighted text</mark>

Links

Links connect your page to others. Use target="_blank" to open in a new tab.

<a href="https://example.com">Visit Example</a>
<a href="page.html" target="_blank">Open in new tab</a>
