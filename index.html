HTML Cheat Sheet

A quick reference for essential HTML elements and best practices.

Headings

Use headings to structure your content from most important (<h1>) to least (<h6>). Only one <h1> per page is recommended.

<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection</h3>

Text & Formatting

Text can be styled using semantic tags for emphasis or structure. Avoid using them only for looks.

<p>This is a paragraph.</p>
<strong>Important text</strong>
<em>Emphasized text</em>
<mark>Highlighted text</mark>

Links

Links connect your page to others. Use target="_blank" to open in a new tab.

<a href="https://example.com">Visit Example</a>
<a href="page.html" target="_blank">Open in new tab</a>

Lists

Lists help organize content. Ordered lists use numbers, unordered lists use bullets.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

Forms

Forms collect user input. Always include labels for accessibility.

<form>
  <label for="name">Name:</label>
  <input type="text" id="name">

  <label for="email">Email:</label>
  <input type="email" id="email">

  <button type="submit">Submit</button>
</form>

Tables

Tables organize data into rows and columns. Always use <thead>, <tbody>, and <th> for better structure and accessibility.

<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Age</th>
      <th>Country</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Ali</td>
      <td>22</td>
      <td>Pakistan</td>
    </tr>
    <tr>
      <td>Sara</td>
      <td>25</td>
      <td>USA</td>
    </tr>
  </tbody>
</table>

Rendered Example:

Name Age Country
Ali 22 Pakistan
Sara 25 USA

Semantic Tags

Semantic tags describe the meaning of content. They make your code easier to read and improve SEO.

<header>Intro content</header>
<nav>Main navigation</nav>
<main>Page content</main>
<footer>Footer content</footer>
